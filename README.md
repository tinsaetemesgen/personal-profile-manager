    ---- Personal Profile Manager ----

This Personal Profile Manager is a React-based web application that allows users to view, manage, and customize their personal profile information through a clean and responsive interface.

This project was developed as a final group project assigned by Elevate Skill, focusing on practical frontend development and team collaboration.


    Features

View and update profile information

Edit personal details (name, bio, contact info)

Theme preference (Light / Dark mode)

Responsive design (mobile-first)

Sidebar dashboard navigation


    Tech Used

React

React Router

Vite

HTML5

CSS

    Installation Guide

Clone the repository:

=> git clone https://github.com/your-username/personal-profile-manager.git

Navigate to the project folder:

=> cd personal-profile-manager

Install dependencies and start the development server:

=>npm install
  npm run dev
